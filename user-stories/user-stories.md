# User Stories

1. As a guest, I want to register an account so I can book properties.
2. As a guest, I want to search for properties by location and date so I can find a place to stay.
3. As a host, I want to create a new property listing so guests can book my place.
4. As a guest, I want to book a property and pay for it so my reservation is confirmed.
5. As a guest, I want to leave a review after my stay so other people know what the place is like.
6. As an Admin, I want to approve listings after they have been uploaded by the host.
