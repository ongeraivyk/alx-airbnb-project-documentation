# Features and Functionalities

This document lists the backend features required for the Airbnb Clone.

Core modules:
- User Authentication (register, login, logout)
- User Management (profile)
- Property Management (create, edit, delete, view)
- Booking System (create, cancel, update)
- Payments (process, confirm, refund)
- Reviews & Ratings (add, edit, delete)
- Notifications (optional)
